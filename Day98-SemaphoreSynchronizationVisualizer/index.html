How Semaphores Work

A semaphore is a synchronization mechanism that controls access to shared resources. It maintains a counter that represents the number of available resources.

  • Wait (P) Operation: Decrements the semaphore. If the value becomes negative, the process is blocked and added to the waiting queue.
  • Signal (V) Operation: Increments the semaphore. If there are waiting processes, one is unblocked and allowed to enter the critical section.
